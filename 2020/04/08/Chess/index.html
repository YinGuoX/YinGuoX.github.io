QUESTION

      根据国际象棋的规则和棋盘大小，使用马的行走规则，将棋盘中的所有小兵都吃完
即通关计分。

界面

ANSWER

Math

关键词 : 抽象、图、DFS、BFS、栈、回溯、哈密顿图
      注 : –来自: 《离散数学引论》
            哈密顿图 : 存在一条哈密顿圈
            哈密顿路 : 图中存在一条经过所有节点并且不重复的路

Think

      主要思想 : 将马和小兵抽象成图上的结点，如果小兵与小兵之间或者小兵与马之间满足马的行走规则,则将其连线，从而形成一张图，进而转换成求图上的哈密顿路问题。

Show Code

//返回结点n的度
int NodeDu(MGraph G,int n){
    int count=0;
    for(int i=0;i<G.numVertexes;i++){
        if(G.arc[n][i]==1){
            count ++;
        }
    }
    return count;
}
void PrintNodeDu(MGraph G){
    for(int i=0;i<G.numVertexes;i++){
        printf("顶点%d的度:%d",i,NodeDu(G,i));
    }
}
