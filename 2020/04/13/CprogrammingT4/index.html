C语言编程T4(妙)

思路:

知规律，且有限，可迭代暴力破解

思路一:回溯

只要将2、3、4、5放在不同位置,满足约束条件即可
又因2,3,4,5有规律,可递增尝试，递减回溯
所以找到回溯判定条件以及相应操作即可
此法较复杂

思路二:穷举(正向思维)

因为只要将2、3、4、5放在四个不同位置，且其有规律
故采用四层或者三层for循环即可遍历所有可能解
所有只要输出满足约束条件的解即可

答案:

核心:
何时回溯、何时穷举！！！

#include <stdio.h>
#include <stdlib.h>
//强行穷举!!!
#include <stdio.h>
#include <stdlib.h>
#define N 6
int main()
{
    int i, j, k, m, n, num;
    int a[N];
    a[0] = 1;
    a[5] = 6;
    num = 0;
    printf("The possible table satisfied above conditions are:\n");
    for (i = 2; i <= 5; i++)
        for (j = 2; j <= 5; j++)
            for (n = 2; n <= 5; n++)
                for (m = 2; m <= 5; m++)
                {
                    a[1] = i;
                    a[2] = j;
                    a[3] = n;
                    a[4] = m;

                    if (a[2] > a[1] && a[4] > a[3] && a[4] > a[1] && a[1] != a[3] && a[2] != a[3] && a[2] != a[4])
                    {
                        num++;
                        printf("\nNo.:%d", num);

                        for (k = 0; k < 6; k++)
                        {
                            if (k % 3 == 0)
                                printf("\n");
                            printf("%3d", a[k]);
                        }

                    }
                }

    return 0;
}
